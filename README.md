# LucidWall

LucidWall is a lightweight, single-purpose Android application designed to give users granular control over their wallpaper blurring. It seamlessly allows you to keep your lock screen crisp and your home screen beautifully blurred—or blur both.

## ✨ Features
* **Image Selection:** Utilizes the native Android Photo Picker for secure, permission-free image selection.
* **Dynamic Blur Control:** Adjust the blur intensity on the fly with a real-time UI preview.
* **Dual Wallpaper Modes:**
    * **Split Mode:** Keep the lock screen image clear while applying the blur to the home screen.
    * **Unified Mode:** Apply the blurred image to both the lock and home screens simultaneously.
* **Optimized Performance:** Uses the official RenderScript Intrinsics Replacement Toolkit for fast, efficient bitmap manipulation.

## 🛠 Tech Stack
* **Language:** Kotlin
* **UI Framework:** Jetpack Compose (Material Design 3)
* **Architecture:** MVVM (Model-View-ViewModel)
* **Asynchrony:** Kotlin Coroutines
* **Image Processing:** Android RenderScript Intrinsics Replacement Toolkit, Coil (for previews)

## 📱 Requirements
* Minimum SDK: 24 (Targeting Android 15 / API 34+)
* Optimized for modern devices like the Google Pixel series.

## 🚀 Getting Started
1. Clone the repository.
2. Open the project in Android Studio.
3. Build and run on a physical device or emulator.
